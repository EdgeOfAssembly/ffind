Function\+: main (client) Purpose\+: Command-\/line client for ffind-\/daemon

Flow\+:
\begin{DoxyEnumerate}
\item Parse command-\/line arguments into search criteria
\item Connect to daemon\textquotesingle{}s Unix socket
\item Send binary protocol message with search parameters
\item Receive results and display with optional colorization
\end{DoxyEnumerate}

Command-\/\+Line Interface\+:
\begin{DoxyItemize}
\item Supports glob patterns, regex, fixed-\/string search
\item Multiple filter types\+: name, path, content, size, mtime, type
\item Context lines\+: -\/A, -\/B, -\/C (like grep)
\item Color control\+: --color=auto\texorpdfstring{$\vert$}{|}always\texorpdfstring{$\vert$}{|}never
\end{DoxyItemize}

Protocol Details\+:
\begin{DoxyItemize}
\item Binary protocol with network byte order (big-\/endian) for integers
\item Variable-\/length strings prefixed with 4-\/byte length
\item Flags byte for case-\/insensitive, regex, content\+\_\+glob
\item See \doxylink{ffind-daemon_8cpp_aa29c4f34a63bacc7d3bcd55e101133fd}{handle\+\_\+client()} in \doxylink{ffind-daemon_8cpp}{ffind-\/daemon.\+cpp} for protocol spec
\end{DoxyItemize}

Error Handling\+:
\begin{DoxyItemize}
\item Uses \doxylink{ffind_8cpp_adb67f2caafe3ae5a6a2a28def0468a9f}{safe\+\_\+write\+\_\+all()} for all socket writes
\item Handles daemon not running gracefully
\item Validates all command-\/line arguments
\end{DoxyItemize}
